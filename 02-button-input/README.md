\# 02 - Button Input



This study focuses on reading a digital input from the onboard user button of the STM32F407G Discovery board.



\## Objective



The objective is to understand GPIO input by reading the onboard user button and controlling an LED according to the button state.



\## Hardware



\- STM32F407G Discovery Board

\- Onboard user button

\- Onboard LEDs



\## Target Behavior



\- When the user button is pressed, the blue LED turns on.

\- When the user button is released, the blue LED turns off.



\## Topics to Learn



\- GPIO Input

\- HAL\_GPIO\_ReadPin

\- Polling

\- Button state

\- Basic input-output control
